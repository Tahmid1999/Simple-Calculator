# Simple-Calculator
A fully working calculator with implementation of postfix.

### Order of operators
1) Parentheses  ()  {}  []
2) Exponents     ^  (If more than one then right to left)
3) Multiplication & Division (Left to Right)
4) Addition & Substraction (Left to Right)
